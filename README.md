# Эмоциональная анатомия британского романа

Shiny-приложение для анализа эмоциональной тональности британских романов с выбором
словаря. Отчётный проект по курсу анализа текста на R (тема «Эмоциональная тональность»).

Репозиторий: https://github.com/Nikita11811/british_sentiment_app.git

## Что делает

На корпусе из 27 британских романов (1740-1876) — Филдинг, Ричардсон, Стерн, Остин,
сёстры Бронте, Диккенс, Элиот, Теккерей, Троллоп — приложение строит:

- эмоциональную арку: текст делится на фрагменты по N слов, для каждого считается тональность;
- профиль восьми эмоций по словарю NRC;
- сравнение трёх словарей на одной книге;
- облако тональных слов и таблицу самых частых эмоциональных слов.

Словари: Bing (плюс/минус), AFINN (оценка от -5 до +5), NRC (восемь эмоций + полярность).

## Как устроено

Тяжёлую токенизацию корпуса сделали заранее (папка `data-raw/`), результат лежит
в `data/` в виде готовых `.rds`. Поэтому приложение запускается быстро: при старте
читает только метаданные и словари, а слова конкретной книги подгружает по запросу.
Все вычисления запускаются по кнопке «Анализировать» (`bindEvent`), а не от каждого
движения мышкой.

```
app.R              приложение (UI + сервер)
data/
  books.rds        список 27 романов
  lexicons.rds     словари bing / afinn / nrc
  tokens/          слова каждой книги по порядку
data-raw/          скрипты подготовки данных (в работе приложения не участвуют)
```

## Запуск

```r
install.packages(c("shiny", "bslib", "tidyverse", "tidytext", "DT", "wordcloud2"))
shiny::runApp()
```

## Подготовка данных заново

Нужен корпус `british_fiction/` (27 txt-файлов). Дальше:

```r
source("data-raw/prep_data.R")        # books.rds + tokens/
source("data-raw/build_lexicons.R")   # lexicons.rds
```
